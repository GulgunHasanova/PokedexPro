# PokedexPro

A modern, comprehensive Pokedex mobile application built with Flutter.

## Features

- **Pokemon Catalog**: Browse through all generations of Pokemon (Gen 1 to Gen 9).
- **Detailed Information**: View stats, types, height, weight, and official artwork for each Pokemon.
- **Team Builder**: Create and manage your own custom Pokemon team of up to 6 Pokemon.
- **Favorites**: Save your favorite Pokemon for quick access.
- **Shiny Mode**: Toggle between normal and shiny forms globally across the app.
- **Compare**: Compare stats between any two Pokemon.
- **Filtering & Search**: Search by name or ID, and filter by region or favorites.

## Technology Stack

- **Framework**: Flutter
- **Language**: Dart
- **API**: [PokeAPI](https://pokeapi.co/)

## Getting Started

1. Ensure you have Flutter installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the application on your connected device or emulator.
